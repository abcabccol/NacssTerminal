_# NACSS Protocol (Neural Action Command Script System)

NACSS is a high-level, action-based scripting language and system protocol designed for high-speed automation and custom system architecture.

## 🚀 Key Features
- **Library Systems**: Supports `W/L` (Global) and `/O/W/L` (Custom User Dictionary).
- **Automation Keys**: Integrated `//:` (NACSSKEY) for verified executions.
- **Persistent Logic**: Built-in support for `/*` (Infinite Loops) and `/*-'` (Auto-Refresh).
- **Node Targeting**: Hierarchical targeting using `OnActionS/N` nodes.

## 🛠 Syntax Guide
| Operator | Function |
| :--- | :--- |
| `//` | Action Trigger |
| `/` | Auto-Adjuster |
| `//:` | NACSSKEY (Automation Key) |
| `/:/:` | Condition Logic |
| `/*` | Infinite Execution |

## 💻 Example Code
`codetype/:/* //NACSS:OnActionS/N:Person:/O/W/L:Action/:("/*/"Ignite:Nexus")`_
